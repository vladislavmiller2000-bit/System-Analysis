# Из состояния Событие Guard [условие] Действие В состояние
1 ● (Initial) пациент записан — создать запись Создан
2 Создан таймер сработал [за 1 день до приёма] отправить уведомление Ожидает подтверждения
3 NEW Создан пациент отменил [до уведомления] удалить запись Отменён
4 Ожидает подтверждения пациент ответил [подтвердил] обновить статус Подтверждён
5 Ожидает подтверждения пациент ответил [отменил] удалить запись Отменён
6 Подтверждён пациент пришёл — начать приём В работе
7 NEW Подтверждён время приёма прошло [пациент не явился] отметить неявку Отменён
8 В работе врач закончил — оформить протокол Завершён

🎯 Все возможные сценарии (пути)

✅ Успешный путь (Happy Path)
Создан → Ожидает подтверждения → Подтверждён → В работе → Завершён → ◉
❌ Отмена #1: Пациент отменил до уведомления
Создан → Отменён → ◉
❌ Отмена #2: Пациент не подтвердил запись
Создан → Ожидает подтверждения → Отменён → ◉
❌ Отмена #3: Пациент подтвердил, но не явился
Создан → Ожидает подтверждения → Подтверждён → Отменён → ◉
