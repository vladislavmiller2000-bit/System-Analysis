# Miro exports (User Story Map)

Сюда можно положить экспорт финальной доски User Story Map из Miro.

## Рекомендуемый набор для портфолио

1) `user-story-map.pdf` — лучший вариант для “читаемости” и печати.
2) `user-story-map.png` (или JPG) — удобно для быстрого предпросмотра прямо в GitHub.

Опционально:
- `user-story-map.csv` — полезно, если хочешь показать структурированные данные (для аналитики/бэклога), но для портфолио обычно вторично.

## Как назвать файлы

Чтобы было аккуратно в репозитории:
- `user-story-map_vita_checkup.pdf`
- `user-story-map_vita_checkup.png`
- `user-story-map_vita_checkup.csv`
