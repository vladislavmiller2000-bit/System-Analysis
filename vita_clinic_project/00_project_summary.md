# 🎯 Сводка проекта: Медицинская клиника "Вита"

## Краткое описание проекта

**Цель:** Автоматизация процесса записи на приём к врачу через мобильное приложение для сети медицинских клиник "Вита" имени Е.Л. Яницкой.

**Ключевая проблема:** Сейчас запись осуществляется только по телефону, что неудобно для современной аудитории (средний возраст ~30 лет) и приводит к потере клиентов.

---

## 📌 Ключевые требования (сводка)

### Топ-5 критических функциональных требований:
1. **Онлайн-запись** — возможность записаться на приём через приложение
2. **Выбор врача** — самостоятельный выбор специалиста
3. **Выбор времени** — просмотр и выбор доступных слотов
4. **Уведомления** — напоминание о приёме и подтверждение
5. **Отмена записи** — возможность отменить запись

### Типы выявленных требований:

| Тип | Количество | Источник |
|-----|------------|----------|
| Функциональные (FR) | 17 | Интервью |
| Нефункциональные (NFR) | 4 | Контекст + интервью |
| Бизнес-требования (BR) | 4 | Описание проекта |
| Пользовательские (UR) | 4 | Интервью |
